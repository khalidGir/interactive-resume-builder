# Interactive Resume Builder - Frontend

This is the frontend application for the Interactive Resume Builder, built with Next.js.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Environment Variables

Copy `.env.local` from `.env.local.example` and update the values as needed.
